F1 App Test Instructions

Testing the F1 News Ticker

  1. Open http://localhost:5173
  2. Scroll to the bottom of the page to see the F1 News Ticker
  3. Click on the red "F1 NEWS" label in the ticker
  4. You should be redirected to the new Motorsport News page
  5. Verify the page displays "Motorsport News" heading and content
  6. You can also use the navigation menu "Motorsport News" link to access the page

Expected Behavior

  • The F1 NEWS label should have a pointer cursor when hovering
  • On hover, the label should brighten slightly and have a subtle glow
  • Clicking the label should navigate to /motorsport-news
  • The new page should load with proper F1 theme styling
  • The ticker should remain visible on all pages
